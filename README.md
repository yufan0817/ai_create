# 🎭 梨园智境（Liyuan AI）

> AI赋能国粹传承，让京剧学习走进新时代。

## 项目简介

梨园智境是一款基于大语言模型（LLM）的沉浸式京剧学习平台，旨在通过人工智能技术降低传统戏曲艺术的学习门槛，让更多年轻人了解、学习并喜爱京剧文化。

项目融合了：

* 🎭 京剧文化传承
* 🤖 AI智能解析
* 🎮 游戏化学习
* 💬 虚拟角色互动

通过“学、玩、问、练”的方式，为用户打造全新的数字京剧学习体验。

---

## 项目愿景

传统京剧艺术拥有数百年的历史积淀，但在数字时代面临：

* 学习门槛较高
* 优质资源分布不均
* 年轻群体接触机会有限
* 非遗文化传播困难

梨园智境希望借助AI技术，构建一个人人都能参与的京剧学习平台，让传统文化以更年轻、更有趣、更智能的方式焕发新的生命力。

---

## 核心功能

### 🎼 AI唱段解析

用户可以：

* 浏览经典京剧唱段库
* 搜索唱段内容
* 输入自定义唱词
* 获取AI智能解析

AI将自动生成：

* 剧目出处
* 人物背景
* 白话文翻译
* 情感分析
* 京剧知识点
* 表演建议

---

### 🎭 角色体验馆

用户可以与经典京剧人物进行沉浸式对话。

支持角色包括：

* 诸葛亮
* 关羽
* 包拯
* 穆桂英
* 杨贵妃
* 武松
* 李逵
* 孙悟空（京剧形象）

基于大模型角色扮演能力，实现个性化实时互动体验。

---

### 🏮 京剧知识闯关

采用游戏化学习模式。

功能包括：

* 随机抽题
* 京剧知识问答
* 行当知识学习
* 脸谱文化认知
* 剧目历史了解

完成挑战后可获得：

* 学习评分
* 能力分析
* 专属称号

例如：

* 梨园学徒
* 梨园新秀
* 梨园雅士
* 梨园名角
* 梨园宗师

## 项目结构

```bash
src/
├── app/
│   ├── chat/          # 角色体验馆
│   ├── learn/         # 唱段解析
│   ├── quiz/          # 知识闯关
│   ├── api/           # AI接口
│   └── page.tsx       # 首页
│
├── components/        # UI组件
├── data/              # 唱段库与题库
└── lib/               # 工具函数
```

## 本地运行

安装依赖：

```bash
npm install
```

启动开发环境：

```bash
npm run dev
```

访问：

```bash
http://localhost:3000
```
